# Data Dictionary

## 1. `dim_users`
| Column Name | Data Type | Description |
| :--- | :--- | :--- |
| `user_id` | STRING | Unique Google Analytics device/cookie identifier. |
| `first_seen_date` | DATE | The date of the user's very first recorded event. |
| `last_seen_date` | DATE | The date of the user's most recent recorded event. |
| `total_sessions` | INTEGER | The total number of unique sessions generated by the user. |
| `lifetime_value_usd` | FLOAT | The total sum of purchase revenue generated by the user. |

## 2. `dim_sessions`
| Column Name | Data Type | Description |
| :--- | :--- | :--- |
| `session_id` | STRING | Primary Key. Concatenation of `user_id` and the GA4 session start timestamp. |
| `user_id` | STRING | Foreign Key linking to `dim_users`. |
| `session_start_time` | TIMESTAMP | The exact timestamp the session began. |
| `session_end_time` | TIMESTAMP | The exact timestamp of the final event in the session. |
| `device_type` | STRING | Hardware category (e.g., mobile, desktop, tablet). |
| `operating_system` | STRING | The OS used during the session (e.g., iOS, Android, Windows). |
| `country` | STRING | Geographic country detected via IP. |
| `city` | STRING | Geographic city detected via IP. |

## 3. `fct_events`
| Column Name | Data Type | Description |
| :--- | :--- | :--- |
| `event_id` | STRING | Primary Key. Cryptographic hash of user ID, timestamp, and event name. |
| `session_id` | STRING | Foreign Key linking to `dim_sessions`. |
| `user_id` | STRING | Foreign Key linking to `dim_users`. |
| `event_time` | TIMESTAMP | Exact microsecond timestamp of the event. |
| `event_name` | STRING | The GA4 event triggered (e.g., `page_view`, `purchase`). |
| `page_url` | STRING | The specific URL where the event occurred. |
| `transaction_id` | STRING | The unique checkout ID (only populated on `purchase` events). |
| `revenue_usd` | FLOAT | The total revenue for the transaction (only populated on `purchase` events). |

## 4. `fct_product_interactions`
| Column Name | Data Type | Description |
| :--- | :--- | :--- |
| `interaction_id` | STRING | Primary Key. Hash of user, timestamp, event, and item ID. |
| `session_id` | STRING | Foreign Key linking to `dim_sessions`. |
| `event_time` | TIMESTAMP | Exact microsecond timestamp of the product interaction. |
| `event_name` | STRING | The action taken on the product (e.g., `add_to_cart`). |
| `item_id` | STRING | The unique SKU/Identifier for the product. |
| `item_name` | STRING | The plain-text name of the product. |
| `item_category` | STRING | The internal merchandising category (e.g., Apparel). |
| `item_price` | FLOAT | The unit price of the item. |
| `item_quantity` | INTEGER | The number of units interacted with or purchased. |
